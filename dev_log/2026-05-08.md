# 2026-05-08 開發紀錄

## SaleDetailPage 資訊架構重設計（參照信義房屋）

**目標：** 將出售詳情頁改版至符合台灣主要房仲平台（信義房屋）的資訊架構，維持現有 M3 暖光視覺風格。

### 後端 (Go)
- `go-service/internal/modules/sale_listing/dto.go`：`PropertySummaryResponse` 新增 9 個欄位：`has_dark_room`、`shared_area`、`awning_area`、`land_area`、`building_structure`、`exterior_material`、`building_usage`、`zoning`、`units_on_floor`。
- `go-service/internal/modules/sale_listing/handler.go`：`toResponse()` 補齊 9 個欄位的 mapping。

### 前端 (TypeScript / React)
- `react-service/src/api/saleListingApi.ts`：`PropertySummary` type 新增 9 個欄位（與 Go DTO 對齊）。
- `react-service/src/pages/SaleDetailPage.tsx`：全面重寫，四個 section：
  - **Hero**：總價大字（萬元 formatWan）+ 建坪單價副標 + chips + 快速統計條（建坪/格局/屋齡/樓層/車位）
  - **基本資料**：面積拆解方程式列（主建物＋附屬建物＋陽台＝合計）+ 地坪 / 共有部份 / 雨遮 / 邊間暗房合併顯示
  - **建物詳情**（新 section）：建物結構 / 外牆建材 / 該層戶數 / 謄本用途 / 使用分區（有資料才顯示）
  - **出售條件**：原有內容保留

### 種子資料
- `infra/init/15-demo-properties.sql`：20 筆 Demo 物件 INSERT 補上 8 個新欄位，並重新注入資料庫。
