### Step 8 — 申請租客憑證

**前往：** `http://localhost:5173/credential/tenant`

| 欄位 | 複製這個值 |
|------|-----------|
| 職業類型 | `上班族` |
| 任職單位 | `科技公司` |
| 月收入區間 | `60,000 以上` |
| 同住人數（含本人） | `2` |
| 同住說明 | `伴侶同住，無小孩。` |
| 搬入時程 | `一個月內` |
| 補充說明 | `作息穩定，偏好安靜社區。` |

→ 點「送出申請」，等待審核通過

### Step 9 — 租屋需求 1（台北大安 / 信義兩房）

| 欄位 | 複製這個值 |

| 目標行政區 | 選「台北市」→「大安區」、「信義區」 |

| 最高預算（月租） | `36000` |
| 格局備註 | `希望兩房以上，有對外窗與陽台。` |
| 預計搬入日 | （一個月後日期） |
| 需要寵物友善 | ✅ 勾選 |
| 需要停車位 | ❌ 不勾 |
| 最小坪數 | `12` |
| 最大坪數 | `28` |
| 最低房數 | `2` |
| 最短租期（月） | `12` |
| 可開伙 | ✅ 勾選 |
| 可設籍 | ✅ 勾選 |
