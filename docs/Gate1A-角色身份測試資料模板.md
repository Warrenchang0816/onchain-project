# Gate 1A 角色身份測試資料模板

更新日期：2026-04-22

這份文件提供 `OWNER / TENANT / AGENT` 三種身份的本地測試資料模板。

## 使用前提醒

- 先完成 `KYC` 並登入，否則角色身份頁不會開放。
- 目前上傳格式建議使用 `PNG` 或 `JPG`，不要先用 `PDF`。
- `holderName` 建議與 KYC 姓名完全一致，否則智能審核容易失敗。
- 目前智能審核只看最低辨識條件，這份模板就是依照目前程式規則設計。
- 若你不想拿真實個資測試，請直接使用這份假資料文字做白底黑字圖片。

## 建議測試方式

1. 用 Word、Google Docs、Canva 或畫圖工具建立白底黑字文件。
2. 把下方文字直接貼進去。
3. 匯出成 `PNG` 或 `JPG`。
4. 在身份頁填表單時，表單內容與圖片文字盡量一致。

## OWNER 屋主身份

### 表單填寫範例

- `holderName`：張期徫
- `propertyAddress`：台北市大安區復興南路一段100號5樓
- `ownershipDocNo`：北市字第123456號

### 主文件圖片文字模板

```text
建物所有權狀

所有權人：張期徫
權利範圍：全部
不動產標示
台北市大安區復興南路一段100號5樓
權狀字號：北市字第123456號
```

### 補充文件可選模板

```text
房屋持有補充說明

所有權人：張期徫
物件地址：台北市大安區復興南路一段100號5樓
說明：本人為該物件現行持有人，提供作為屋主身份認證測試資料。
```

### 這組資料為什麼容易通過

- 有 `建物所有權狀 / 所有權人 / 權利範圍 / 不動產` 這類屋主關鍵字
- 有姓名 `張期徫`
- 表單有填 `propertyAddress`

## TENANT 租客身份

### 表單填寫範例

- `holderName`：張期徫
- `employerOrSchool`：屋柱科技股份有限公司
- `incomeHint`：每月固定薪資 60000 元

### 主文件圖片文字模板

```text
在職證明

姓名：張期徫
公司：屋柱科技股份有限公司
職稱：產品經理
薪資：每月新台幣 60000 元
```

### 備用主文件模板

```text
薪資證明

姓名：張期徫
所得資料
每月固定薪資 60000 元
```

### 補充文件可選模板

```text
租住能力補充說明

姓名：張期徫
工作單位：屋柱科技股份有限公司
收入說明：每月固定薪資 60000 元
說明：以上資料僅用於租客身份認證測試。
```

### 這組資料為什麼容易通過

- 有 `薪資 / 在職 / 所得` 這類租客能力關鍵字
- 有姓名 `張期徫`
- 表單與圖片內容一致

## AGENT 仲介身份

### 表單填寫範例

- `holderName`：張期徫
- `licenseNumber`：ABC123456
- `brokerageName`：屋柱安心仲介

### 主文件圖片文字模板

```text
不動產營業員證照

姓名：張期徫
登錄字號：ABC123456
服務單位：屋柱安心仲介
```

### 備用主文件模板

```text
不動產經紀人證照

姓名：張期徫
證照字號：ABC123456
執業單位：屋柱安心仲介
```

### 補充文件可選模板

```text
仲介身份補充說明

姓名：張期徫
證照或登錄字號：ABC123456
所屬公司：屋柱安心仲介
說明：以上資料作為仲介身份認證測試用途。
```

### 這組資料為什麼容易通過

- 有 `不動產經紀 / 營業員 / 經紀人 / 登錄字號 / 證照` 這類仲介關鍵字
- 有姓名 `張期徫`
- 圖片內的 `ABC123456` 與表單 `licenseNumber` 完全一致

## 故意測失敗的版本

如果你想驗證「不通過」流程，可以用下面方式改一個條件就好。

### OWNER 失敗版

- 把圖片裡姓名改成 `陳小華`
- 或把圖片文字改到完全沒有 `建物所有權狀 / 所有權人 / 權利範圍 / 不動產`

### TENANT 失敗版

- 保留姓名，但把圖片改成一般自述，不放 `薪資 / 在職 / 所得 / 勞保 / 扣繳`

### AGENT 失敗版

- 表單填 `ABC123456`
- 圖片改成 `ABC654321`

## 建議測試順序

1. 先測 `AGENT`
2. 再測 `OWNER`
3. 最後測 `TENANT`

原因：

- `AGENT` 的規則最明確，最容易看出智能審核有沒有正常運作
- `OWNER` 次之
- `TENANT` 關鍵字最容易受文件格式影響
